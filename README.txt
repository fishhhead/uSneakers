# uSneakers

Author: Yufeng Zhu,  SID:218055631

Project Overview:
The application I develop is a comprehensive introduction for latest sneakers and gallery.
The introduction will show what this app is about, and gallery will display featured sneakers with animation.
There is a coupon generator for range of sneakers. The coupon user can get and use it when they purchase sneaker office shopping website.
It also offers a service to verify whether a sneaker is official or fake.
Use bought this app and registered will be charge free for bring their sneaker to our verification center, which is shown on Google Map.
The user will be required to login to use this application. They will be able to edit their personal information.
